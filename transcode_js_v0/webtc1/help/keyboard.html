Virtual Keyboard

By pressing the virtual keyboard image , the user toggles a window whose appearance depends on the 'Keyboard input' Preference. There are several features common to all the virtual keyboard windows:

  • A drag bar: By pressing and holding the left mouse button when the mouse pointer is in the drag bar, the user may reposition the virtual keyboard.
  • A Clear button: By clicking this, the 'Input Field' is emptied.
  • An X close button: By clicking this, the virtual keyboard is closed. The virtual keyboard may also be closed by click the virtual keyboard image .
  • Other keys Pressing other keys generally adds appropriate information into the Input Field.

The virtual keyboard also may provide the user a quick reference regarding her 'Keyboard input' Preference.
