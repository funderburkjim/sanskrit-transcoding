Word List

The Word List panel to the left of the display provides some context for the word whose main definition appears in the larger Word Definition Area. When the user 'enters' a word in the Input Field, the server provides a definition for the word for display in the Word Definition Area and provides a list of nearby words in the Word List panel. The particular choice of nearby words depends is either hierarchical or alphabetical, according to the setting of the 'List option' in the Preferences.

The word appearing in the Input Field is shown in a different color in the word list.

If the user clicks a word in the Word List, then the definition of that word is shown in the Word Definition Area.

If the user clicks the ▲ element, then a new word list is displayed; In this new list, the word below the ▲ in the former list appears as the last word, in a different color, and the other words are those preceding this word. The word in the Input Field, and the word shown in the Word Definition Area do not change.

Similarly, if the user clicks the ▼ element, a list of following words is shown in the Word List area.
