Help

This display of the Monier-Williams Sanskrit-English Dictionary was developed in response to suggestions by users. While using the same data as the previous (2008) version, this 2011 version has a new way of specifiying input/output options and permits users to, in effect, type Devanagari. This version also incorporates a list of words 'near' a given word, and has some limited ability to permit the user to get information even in absence of knowing the exact spelling of a word.

The web page may be considered to have several parts.

  • The dictionary title, bracketed by images identifying the educational institutions, Cologne University and Brown University, by whose cooperation the underlying data and the display technologies were developed.
  • A row of Links to elsewhere on the Cologne Sanskrit-Lexicon website.
  • A Preferences Button by which the user may select various options pertaining to either input or output.
  • A Word Input Field in which is displayed the word for which the user wishes to display a definition. The user may enter this word by typing keys (according to the Preferences chosen). Alternatively, the user may use the mouse to click appropriate keys on the 'virtual keyboard' which is availabe by clicking the Keyboard Button.
  • A Word Definition Area displays a definition of the word shown in the Word Input Field. A list of words that are 'near' the word is shown in the Word List area.
