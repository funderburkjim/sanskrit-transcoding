System Requirements

We have collected here what we know about the behavior of this dictionary display in various browsers and operating system environments. Our development enviornment is the latest version of Mozilla FireFox browser for the Windows (Vista/7) operating system; so, we think everything works in this environment.

First, you will have to enable Javascript in your browser. If you want the convenience of inter-session persistence of Preferences, you will have to permit Cookies. If you normally don't permit cookies, you can probably allow cookies from our url 'sanskrit-lexicon.uni-koeln.de'.

'Phonetic' keyboard input seems works fine in most browsers.

The 'Unicode' keyboard input has been subject to less scrutiny by us.

If you notice specific computer-browser problems, please communicate the details of the problem to us via email. Alternatively, if possible, try another browser known to work.
