Input Field

The input field serves two main functions:

  • It displays the user input. When the text insertion point is within this field, the user's key strokes are displayed on the basis of the Preferences for 'Keyboard input' and 'Input display'. When the user has enabled the virtual keyboard, then mouse clicks therein are similarly displayed in the input field.
  • When the input field shows the spelling of the word of interest to the user, then the user may generate a request to the server for a definition of this word. This request is initiated by pressing the 'Enter' key while the text insertion point is in the input field.
