Preferences

The Preferences Button causes the opening of a new window, in which various options may be set to control how the input is interpreted and displayed in the Word Input Field, and how server information is display in the Word Definition area and in the Word List area. These options also select details of the virtual Keyboard, whose display and usage is optional.

For those familiar with the previous Monier Williams display, it may be useful to think of the 'Keyboard input' and 'Input display' settings described below as functional replacements for the 'input' and 'output' settings of the previous display. However, the two systems are not identical.

For MW 2011, there is an additional List option which controls the manner in which the Word List is generated.

Browser 'Cookies' are used to save the Preferences chosen. This is convenient in that the user's choice in one browser session are thus available in the next session; there is no need to set the Preferences again. However, users who have disabled cookies in their browser will need to set Preferences each session.

A detailed explanation of the options is now attempted.

  • Keyboard input: The user chooses one of two systems of input, identified as Phonetic or Unicode. Once this choice is made, the user chooses one of several appropriate sub-options. The choices made are used in conjunction with the Input Display choice to govern how the user keystrokes are presented in Word Input Field and how information from the server is displayed in the Word Definition Area and the Word List Area.
    • Phonetic: The Phonetic transliteration schemes represent the letters of the Sanskrit alphabet by means of certain of the 'standard' ASCII characters. When one of these schemes is chosen, the virtual Keyboard can be used as a reference to the representation. The supported transliterations are:

  • Input display: The choices in this list determine how the user input in the Input Field are formed in relation to the the Keyboard input preference.
    • Phonetic: The user may choose that displayed Sanskrit is either
      • the same as keyboard input
      • in standard Romanization
      • in Devanagari
    • Unicode: The user has no choice of display; rather, the display is set to correspond to the Keyboard input Unicode preference,namely
      • Roman Unicode is displayed in standard Romanization
      • Devanagari INSCRIPT is displayed in Devanagari
      • Devanagari QWERTY is displayed in Devanagari
  • Server display: As the combination of preferences for 'Keyboard input' and 'Input display' determine the display of Sanskrit by the Browser in the Input Field, so the combination of preferences for 'Keyboard input' and 'Server display' determine the display of Sanskrit generated by the server in the Word Definition and Word List areas.
    In the current implementation, the choices of the Server display cannot be changed by the user; in all cases, the choices of 'Input display' are the ones used for Server display.
    Tecnical note: For Phonetic Keyboard input, the value shown for the 'Server display' may differ from that for the 'Input display'. Please ignore this difference.
  • List option: For the Monier Williams Dictionary, we have provided two ways to generate and display the list of words in the Word List area. The list option preference determines which way is used.
    • The hierarchical list displays the words in the order of appearance in the dictionary. This list is generally in alphabetical order, but many deviations from alphabetical order occur.
      Further, we distinguish by indentation the typographically distinguished 'four mutually correlated lines' of words in the dictionary (see Page xiv of the Preface in the scanned edition for the description by Monier-Williams).
      Words without indentation correspond to the 'main line', words with one place of indentation correspond to the 'subordinate line', and so forth.
    • The alphabetical list displays the words from the dictionary in Sanskrit alphabetical order. The list includes words appearing in the 'Additions and Corrections' section of the printed dictionary (starting on page 1308). It also includes words which, while not appearing as headwords in the dictionary, are referred to in the definitional text of other headwords; we refer to these as 'parenthetical head words'.
